laravel 的安装需要借助 composer ，百度一下，安装，然后去 GitHub 下载 laravel

切换到项目文件夹 ，在不选中任何文件的前提下按住 shift + 鼠标右键，打开 Powershell 或者 cmd

输入：composer install
