#### 第一步

先看看 npm 和 node 的版本：
```shell
node -v
npm -v
```

两条命令都有输出说明本地环境是 ok 的，可以安装 Electron 了。

#### 安装 Electron

```shell
mkdir electron-app
cd electron-app/
npm init
npm install electron --save-dev
```

慢是它一贯的作风，等就对了。
